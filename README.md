一个想法，用于模拟单细胞生物演化
- 用类表示简单的单细胞类cell
- 某个类有生命值blood和遗传物质dna
- 遗传物质dna为二进制代码，其首端若干二进制代表其生物信息，编码0000为cif，编码0001为celse，编码0010为cdo。编码0011为cgoto。
- 空间最大容纳100个细胞，细胞生命值超过100繁殖为两个细胞，繁殖时一个子细胞dna随机突变。细胞随机碰撞，相遇时dna编译运行。
